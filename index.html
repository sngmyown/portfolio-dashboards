Portfolio Dashboards

투자 계좌 대시보드

장기투자 계좌와 커버드콜 배당 계좌를 분리해서 관리합니다. 입력 데이터는 같은 브라우저의 로컬 저장소에 자동 저장되며, 각 대시보드에서 JSON으로 내보내기/불러오기가 가능합니다.

중요: ZIP 안에서 바로 index.html을 열면 아래 링크가 작동하지 않을 수 있습니다. 반드시 ZIP을 압축 해제한 뒤, 같은 폴더 안에 index.html, longterm-dashboard.html, covered-call-dashboard.html 3개가 함께 있는 상태에서 실행하세요.
LONG TERM

장기투자 대시보드

포지션, 현금, 월간 체크리스트, 시나리오 메모를 관리합니다. JSON 백업/복원 버튼이 포함되어 있습니다.

 COVERED CALL

커버드콜 배당 대시보드

월별 배당 수령액, FCF 적립액, 퀀트 점프 기록을 관리합니다. 자동저장 및 JSON 백업/복원 기능이 포함되어 있습니다.
저장 방식: 같은 웹사이트 주소 + 같은 브라우저에서는 자동 저장됩니다. 브라우저를 바꾸거나 다른 기기에서 쓰려면 각 대시보드의 내보내기로 JSON 백업 파일을 만든 뒤, 다른 기기에서 불러오기로 복원하면 됩니다.
