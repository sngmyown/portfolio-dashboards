# Portfolio Dashboards

## 실행 방법

1. ZIP 파일을 먼저 압축 해제합니다.
2. 압축 해제된 폴더 안에 아래 3개 파일이 같은 위치에 있어야 합니다.

```text
index.html
longterm-dashboard.html
covered-call-dashboard.html
```

3. `index.html`을 브라우저로 엽니다.
4. 장기투자 대시보드 또는 커버드콜 배당 대시보드를 클릭합니다.

## 중요한 점

ZIP 압축파일 내부에서 `index.html`을 바로 더블클릭하면 브라우저가 임시 폴더에서 파일을 열 수 있습니다.
이 경우 상대 링크가 깨져서 각 대시보드를 클릭해도 열리지 않을 수 있습니다.

반드시 압축을 푼 뒤 실행하세요.

## 저장 방식

- 같은 브라우저 + 같은 주소에서는 `localStorage`에 자동 저장됩니다.
- 다른 기기나 브라우저로 옮길 때는 각 대시보드의 `내보내기` 버튼으로 JSON 백업 파일을 만들고, `불러오기`로 복원하세요.
